# Google Sheets 템플릿 구조

## Sheet 1: artist_list

### 컬럼 구조
| 컬럼 | 설명 | 예시 |
|------|------|------|
| A: 구분 | 수상 또는 초청 구분 | 수상작가, 초청작가 |
| B: 분류 | 작가 분류 | 판매 우수, 활동 우수, IPUS |
| C: 부문 | 작품 부문 | 가죽공예, 도자기, 목공예 |
| D: URL | 작가 홈페이지 | https://www.idus.com/w/artist/... |
| E: 작가명 | 작가 이름 | 아비엥또 |
| F: 작가 UUID | 고유 식별자 | a13ad97f-8917-469f-867b-d19c2d6dd5c1 |

### 데이터 예시
```
구분       | 분류      | 부문    | URL                                   | 작가명    | 작가 UUID
수상작가   | 판매 우수 | 가죽공예 | https://www.idus.com/w/artist/abc... | 아비엥또  | a13ad97f-8917-469f-867b-d19c2d6dd5c1
초청작가   |           | 도자기   | https://www.idus.com/w/artist/def... | 여누즈    | 25cee04d-d3f9-48d5-9fce-19fd0e4476b7
수상작가   | 활동 우수 | 목공예   | https://www.idus.com/w/artist/ghi... | 45번길쿠키 | 5d461746-da18-4bff-a4ff-4323b04cd20c
```

## Sheet 2: table_info

| 컬럼 | 설명 | 예시 |
|------|------|------|
| A: 테이블 | 테이블 번호 | 작가 테이블 1 |

## 데이터 입력 가이드

### 1. 기존 CSV 데이터 가져오기

**artist_list.csv → Sheet1에 붙여넣기:**
1. `artist_list.csv` 파일을 엑셀이나 텍스트 에디터로 열기
2. 모든 내용 복사 (Ctrl+A, Ctrl+C)
3. Google Sheets에서 Sheet1의 A1 셀 클릭
4. 붙여넣기 (Ctrl+V)
5. 데이터가 열로 올바르게 분리되었는지 확인

**table_info.csv → Sheet2에 붙여넣기:**
1. `table_info.csv` 파일을 엑셀이나 텍스트 에디터로 열기
2. 모든 내용 복사 (Ctrl+A, Ctrl+C)
3. Google Sheets에서 Sheet2의 A1 셀 클릭
4. 붙여넣기 (Ctrl+V)
5. 데이터가 열로 올바르게 분리되었는지 확인

### 2. 수동으로 데이터 입력

**새 작가 추가 (artist_list):**
1. Sheet1의 마지막 행 다음 셀 클릭
2. 각 열에 해당하는 정보 입력
3. 작가 UUID는 UUID 생성기로 생성하거나 임의의 고유 문자열 사용

**테이블 배정 (table_info):**
1. Sheet2에서 같은 테이블의 작가들은 테이블, 테요1, 테요2를 동일하게 입력
2. 작가 UUID는 Sheet1의 해당 작가 UUID와 정확히 일치해야 함

### 3. 데이터 검증

**필수 항목:**
- artist_list: 작가명, 작가 UUID
- table_info: 테이블, 테요1, 테요2, 작가, 작가 UUID

**주의사항:**
- 작가 UUID는 두 시트에서 정확히 일치해야 합니다
- 테이블 이름은 일관성 있게 입력 (예: "작가 테이블 1", "작가 테이블 2", ...)
- 첫 번째 행은 반드시 헤더(컬럼명)여야 합니다
