[강의 제목] 비개발자가 코딩 에이전트(Antigravity)를 업무에 활용하는 방법

[강의 목적] IDE 기반의 AI 코딩 에이전트에 대해 소개하고 AI 에이전트를 활용하여 비개발자가 코딩을 하지 않고 코딩 에이전트를 이용해 본인의 업무를 효과적으로 수행할 수 있는 방법에 대해 3가지 사례를 통해 소개합니다.

[주요 코딩 에이전트 소개]
- Google Antigravity
- Cursor

[주요 활용 사례]
1. 주차등록 자동화 시스템 사례 소개
 - 코딩 에이전트 기반으로 구글시트, 구글앱스스스크립트, playwright, slack bot, 사내 서버를 이용해 주차등록 자동화 시스템 구현
 - 시스템의 동작 원리 (개념도)
 - 각 툴의 역할과 코딩 에이전트의 역할
 - 사내 서버 이용을 위해서는 Devops셀과 사전 논의 필수
2. 발표 자료 슬라이드 웹페이지 만들기
 - 발표에 필요한 데이터와 요구사항을 기반으로 코딩 에이전트에게 발표 자료 만들게 하기
3. HMA 테이블 요정 어시스턴트 만들기
 - 정리된 데이터와 코딩 에이전트를 이용하여 원하는 형태로 웹페이지 만들기
 - Antigravity와 실제 화면을 보면서 필요한 요구사항을 다듬고 완성되면 GAS로 서버 걱정없이 웹페이지화 하기


[핵심 메시지]
1. 코딩 에이전트를 실제 업무에 활용하기 위해서는 무엇이 필요하고 어떤 모습으로 구현되었으면 하는지에 대해 명확히 문서화할 수 있어야 함.
2. 비개발자가 코딩에이전트를 잘 활용하기 위해서 코드베이스가 존재하면 큰 도움이 됨. 구글시트 데이터, 노션 페이지의 정보는 모두 하나의 코드베이스, 데이터베이스인 만큼 이를 잘 활용하면 훨씬 더 효율적으로 에이전트를 활용할 수 있음
3. 코딩에이전트가 내장하고 있는 브라우저 기능이나 playwright을 활용해서 사람이 웹페이지에서 할 수 있는 거의 모든 일을 자동화할 수 있음. 무엇이 되고 무엇이 안되는지에 대한 감을 익히는 것이 핵심이며, 그 감은 직접 해보면서 익힐 수 있음
